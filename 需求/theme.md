# 🎨 终端主题配色设计文档

本终端应用提供多套视觉主题，以满足不同使用场景（开发、长时间使用、展示、炫酷风格等）。每套主题均包含：背景色、文字色、光标色以及状态颜色体系。

---

# 🌙 主题一：专业深色（Dark Pro）

**定位：默认主题 / 开发首选**

**特点：**

* 深灰背景，降低视觉疲劳
* 高对比度文字，清晰易读
* 配色接近主流开发工具（如 VSCode）

**适用场景：**

* 日常开发
* 长时间终端使用
* SSH远程操作

**视觉感受：**

* 稳重、专业、耐看

---

# 🌌 主题二：赛博科技（Cyber Neon）

**定位：科技感 / AI终端推荐主题**

**特点：**

* 深蓝背景 + 霓虹色高亮
* 青色/紫色为主色调
* 对比强烈，视觉冲击感强

**适用场景：**

* AI功能展示
* 产品演示
* 高级终端界面

**视觉感受：**

* 未来感、科技感、炫酷

---

# 🌙 主题三：柔和夜间（Soft Night）

**定位：护眼主题**

**特点：**

* 低饱和度配色
* 减少强对比刺激
* 色彩柔和不过亮

**适用场景：**

* 长时间使用（尤其夜间）
* 编程 / 写日志 / 阅读输出

**视觉感受：**

* 舒适、安静、不刺眼

---

# ☀️ 主题四：浅色简洁（Light Clean）

**定位：白天使用 / 清爽风格**

**特点：**

* 白色背景
* 深色文字
* 整体干净简洁

**适用场景：**

* 白天办公环境
* 与系统浅色模式统一

**视觉感受：**

* 清爽、简洁、轻量

---

# 🧪 主题五：黑客矩阵（Matrix）

**定位：个性化 / 展示用**

**特点：**

* 黑底 + 绿色文字
* 高统一色调
* 强烈“终端感”

**适用场景：**

* 产品展示
* 个性化用户
* 黑客风格 UI

**视觉感受：**

* 极客、复古、沉浸感强

---

# 🎯 状态颜色规范（建议统一）

为了提升终端的可读性与专业度，建议统一使用以下语义颜色：

| 状态类型 | 建议颜色  | 含义        |
| ---- | ----- | --------- |
| 成功   | 绿色    | 操作成功 / 正常 |
| 错误   | 红色    | 报错 / 失败   |
| 警告   | 黄色    | 风险提示      |
| 信息   | 蓝色/青色 | 普通提示      |
| AI输出 | 青色/紫色 | AI生成内容    |
