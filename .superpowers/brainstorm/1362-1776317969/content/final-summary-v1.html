设计组件清单

1. 终端主体区域

  • 边框：半透明 Cyan (#00ffff @ 30% opacity) + 微外发光
  • 标题栏：渐变背景 (Cyan→Magenta @ 10% opacity)
  • 状态指示灯：红/绿发光小球
  • 命令提示符：Cyan/Magenta 双色 + text-shadow
  • 输出文字：Green + 柔和发光

2. 空态引导层

  • Logo：64px 圆角方块，渐变背景 + 外发光
  • 标题：Cyan + Magenta 双色拼接，letter-spacing: 2px
  • 分隔线：三色渐变 (transparent → Cyan → Magenta → transparent)
  • 操作卡片：每项独立配色边框 (Cyan/Magenta/Green)
  • 快捷键：统一 Cyan 边框的 kbd 样式

3. 系统监控区

  • 卡片边框：Cyan @ 20% + 微外发光
  • 状态指示：Green 发光小球 + "ONLINE" 文字
  • 进度条：Cyan→Magenta 渐变 + 发光效果
  • 数据分列：CPU Cyan / RAM Magenta 区分
  • 数值：monospace 字体 + text-shadow 微光

实现要点

  • CSS 变量定义：--cyber-cyan, --cyber-magenta, --cyber-green, --cyber-red
  • text-shadow 统一：0 0 8px rgba(color, 0.5)
  • box-shadow 卡片：0 0 20px rgba(color, 0.05)
  • 进度条动画：可选的发光脉冲效果
  • 保持可读性：发光效果不要过度干扰文字识别
